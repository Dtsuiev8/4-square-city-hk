## 📖 項目簡介
**《香港人嘅四方城》** 係一款專為香港人打造嘅地道麻雀手遊。結合咗「正宗港麻邏輯」同埋首創嘅「18 區爭霸戰」系統，旨在建立一個充滿地區歸屬感同埋廣東話文化嘅競技平台。

## 🚀 核心功能 (Core Features)
- **18 區爭霸系統 (District Warfare)**：玩家代表所屬地區出戰，贏取積分提升地區排名。
- **正宗港式邏輯**：支援 3 番起糊、跑馬仔、三人麻（去萬子）等玩法。
- **地道語音系統**：內置廣東話垃圾話配音，還原大牌檔打牌氣氛。
- **大亨 VIP 體系**：專屬特效、金幣加成及地區話事人特權。

## 🛠️ 技術架構 (Technical Stack)
- **Frontend**: Unity 2022.3 LTS (C#)
- **Backend**: Node.js + Express
- **Real-time**: Socket.io (WebSocket)
- **Database**: MongoDB (User Data) + Redis (Matchmaking Cache)
- **Security**: JWT Authentication + Bcrypt Password Hashing
